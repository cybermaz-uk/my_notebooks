## 📂 Repository Contents

This repository contains Jupyter notebooks completed as part of practical lab exercises focusing on text processing and log analysis using Linux command-line tools.

### Included Files:

* `SFS awk 1.ipynb`
  Covers the use of **awk** for structured text processing and data extraction.

* `SFS grep.ipynb`
  Covers the use of **grep** for pattern searching and filtering within files and logs.

---

## 🎯 Learning Objectives

* Understand how to process and extract data using `awk`
* Use `grep` for searching patterns in text files
* Apply scripting techniques to real-world security scenarios
* Analyse and filter system/log data efficiently
